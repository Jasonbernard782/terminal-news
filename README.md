# Terminal News Feed
A calm, terminal-native news reader for people who live in the terminal and don’t want to context-switch.

## Features

- Full-screen terminal UI (Textual)
- Keyboard-first navigation (vim-style)
- Grouped RSS feeds
- Clean reader view inside the terminal
- Open articles in your browser (o)
- Local caching for instant startup

##Keybindings

| Key	          | Action                  |
|-------------- | ----------------------- |
| `j` / `↓`     |	Next item               |
| `k` / `↑`     |	Previous item           |
| `l` / `Enter`	| Open article            |
| `h`	          | Back to list            |
| `o`	          | Open article in browser |
| `r`          	| Refresh feeds           |
| `q`       	  | Quit                    |

## Installation

### Requirements

- Python 3.10+
- macOS or Linux
- A terminal you enjoy using

### Command

pipx install git+https://github.com/jasonbernard782/terminal-news.git

### Run

terminal-news
